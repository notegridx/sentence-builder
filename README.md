# Sentence Builder

**Sentence Builder** は、英語の語順や文構造に慣れることを目的とした、ブラウザベースの学習ツールです。

表示された日本語の文をヒントに、単語を正しい順序で並べ替えて英文を完成させます。  
単語の並びを自ら選択することで、英文の語順や構造を意識的に確認できる学習体験を意図しています。

## Demo

https://notegridx.github.io/sentence-builder/

---

## 想定用途

- IT 技術者が業務で使用する英語表現の学習
- 英文の語順・構造のトレーニング
- PC やタブレット上で完結する軽量な学習
- JSON データを差し替えた独自教材での学習

---

## 特徴

- **単語並べ替え型の学習**
  - 単語をクリックして英文を構築
  - 文構造や語順を能動的に確認できる設計

- **即時フィードバック**
  - 全単語選択時に自動判定
  - 正解時は英文を音声で読み上げ

- **ヒント機能**
  - 同じ問題を 2 回間違えると正解を表示
  - 正しい語順を確認しながら再入力が可能

- **カテゴリ切り替え**
  - フレーズはカテゴリ単位で管理
  - 出題範囲をセレクタで切り替え可能

- **レスポンシブ対応**
  - モバイルでは単語バンクを下部ドックとして表示
  - PC / スマートフォンの両方で利用可能

---

## 収録データ（phrases.json）

このリポジトリに含まれる `phrases.json` には、  
**IT 技術者が仕事で使用する英語表現**を中心に、以下の構成で例文を収録しています。

- 10 カテゴリ  
- 各カテゴリ 20 例文（合計 200 例文）

カテゴリ例：

- 会議  
- 進捗・状況報告  
- 技術・設計  
- トラブル対応  
- レビュー  
- 意思決定  
- ビジネス  
- 調整・依頼  
- 雑談  
- クッション表現  

例文データ形式：

    {
      "id": "m001",
      "category": "会議",
      "ja": "会議を始めましょう。",
      "text": "Let's get started with the meeting."
    }

---

## 拡張性

`phrases.json` を入れ替えることで、

- 別分野の英語例文
- 試験対策用フレーズ
- 英語以外の言語学習データ

などにもそのまま利用できます。

アプリケーション側は JSON の構造にのみ依存しており、  
ロジックを変更せずに教材を差し替えられる設計になっています。

---

## 使い方

1. 日本語の文を確認  
2. 単語バンクから順番に単語をクリック  
3. すべて選択すると自動で正誤判定  
4. 正解すると次の問題へ進みます  

### キーボード操作

| 操作 | 内容 |
|---|---|
| Backspace | 最後に選んだ単語を戻す |
| Esc | 回答をリセット |
| → / ← | 次の問題 / 前の問題 |
